시작
스토리 (배경설명)
(모든 스테이지는 이전 스테이지를 플레이한 적이 있어야 플레이 가능)

1스테이지 (자동시작) + 게임 튜토리얼(스토리) (게임의 조작법과 기본적인 룰을 알려줌)
가게 튜토리얼 + 스토리 (가게 운영 방법(조작법)과 현재 목표를 알려줌)
2스테이지 (가게화면 팝업으로 게임 시작 + 맵 생김새 + 맵 정보 알려줌)
+ 추가 시스템 튜토리얼(스토리) (버닝 시스템과 나중에 업그레이드하면 좋아진다는것을 스토리로 암시)

스테이지 해금 기준은 가게 발전상황에 따라서

돈 벌기 클릭 (스테이지 선택 입장)
스테이지 선택 (정보확인)
게임시작 클릭 (로딩 - > 게임 시작)
게임종료 (번 돈과 재료를 얼마나 썼는지 요약 화면 보여주고 확인 버튼 생김)
확인 클릭 (로딩 - > 가게화면으로 돌아감)

3,4반복
스토리 (를 통해서 빚을 다 갚아야 5스테이지 플레이가 가능하다고 알리고 1~4를 반복 플레이하도록 함) 

5스테이지 - 빚을 다 갚아야 플레이 가능
(롤오버로 빚을 다 갚아야 가능하다고 알려줌)
5스테이지 끝나면
스토리 (를 통해서 재료를 이제부터 구매해서 사용해야한다고 알림 + 구매 방법 튜토리얼)

6스테이지부터 재료를 구매해야함
(여기부터 맵 생김새 뜨는 정보확인창 (게임 시작 클릭화면)에서 현재 갖고있는 재료도 확인하고 갈 수 있도록 재료도 볼 수 있게 추가)
6스테이지 해금부터 가게 청소에 돈을 쓰기 시작 (스토리 + 새로운 목표 제시)

7,8 반복
스토리 (청소를 다 하기 전까진 가게를 늘리지 않을거야!)

9스테이지 - 가게 청소가 다 끝나야 플레이 가능
(롤오버로 가게 청소를 다 해야 플레이 가능하다고 알림)
9스테이지 끝나면
스토리 (이제 가게를 증축시킬 여유가 생겼어! 재료 상자를 해금할 수 있다고 알림 + 업그레이드 창(가게, 직원 나뉘어있음) 튜토리얼)
새로운 목표 제시 (재료 상자 해금 = 가게 발전)

이후로 계속 스테이지 반복 (해금 기준 XX)
하지만 10스테이지부터 시작할 때 (PlayerPrefs로 재료, 소스 해금 어디까지 되었는지)조건 판별해야함
(판별해서 해금된 만큼 UI로 보여줌 + 그 상자에 기능 생김 + 그 재료가 들어간 주문 들어옴)

모든 재료 상자와 소스통이 해금되었다면 (= 해금 순서대로 해야하므로 마지막 소스통3이 해금되었다면)
스토리 (이제 직원을 고용할 여유가 생겼어! 새로운 기능(직원 고용 = 버프) 해금을 알림)
(직원 고용 방법 튜토리얼, 장착 방법 알림, 직원 업그레이드 튜토리얼)
